# Croma Trio – Website

Official website for the jazz ensemble **Croma Trio**.
This project showcases the band’s music, upcoming concerts, and artistic identity through a clean and responsive web interface.

## Overview

This website was designed and developed to provide:

* A central place for listeners to discover Croma Trio
* An up-to-date concert listing
* A platform to connect the trio’s music with its literary influences
* A visually coherent identity aligned with the trio’s artistic direction

A key aspect of the project was to reflect the interdisciplinary nature of the band’s work, linking composition, storytelling, and performance within a single digital space.

## Features

* **Responsive design**

    Optimized for desktop, tablet, and mobile devices.

* **Dynamic multilingual system (EN / ES / CA)**

    The website supports English, Spanish, and Catalan through a custom-built translation system.
    Language switching happens instantly without page reload, dynamically replacing text content for a smooth and uninterrupted user experience.

* **Concerts / Events section**

    Dynamically displays upcoming performances (integrated with an external data source for easy updates).

* **Short stories section**

    Dedicated space for literary texts written by the band’s leader.
    These stories serve as inspirational material for the trio’s compositions, creating a direct link between narrative and music.

* **Media section**

    Showcases audio, video, or visual content related to the trio.

* **Clean UI/UX**

    Minimalist design focused on readability, atmosphere, and artistic coherence.

* **Easy content updates**

    Event data and textual content can be updated without modifying the core code (e.g. via external sources like Google Sheets).

## Tech Stack

* **HTML5** – structure
* **CSS3** – layout and styling
* **JavaScript (Vanilla)** – interactivity, dynamic content, and translation system
* **Google Sheets API** – event management

## Purpose

This project was created to give Croma Trio a professional online presence while remaining lightweight, maintainable, and easy to update.
Particular attention was given to fluid multilingual navigation and the integration of literary content as a compositional foundation, reflecting the ensemble’s artistic identity.
